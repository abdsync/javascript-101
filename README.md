## Description 
This repository contains the javascript code i wrote during my learning where i covered the core concepts and eventually delved into es6 and dom manipulation. 

## Structure 
* `/book-cheatsheets` This repo conaints the resources to learn javascript.
* `/core-js` This repo conatins the code for problems solved using fundamental javascript concepts such as arrays, loops and objects.
* `/dom` This repo focuses on dynamically changing a webpage's content, structure, or style.
* `/es6` This repo focuses on the javascript es6 syntax. 
* `/jquery` This repo focuses on the basics of jquery, a lightweight javascript library.
